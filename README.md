MailGenie

MailGenie is a Chrome Extension that helps users generate professional email replies automatically using AI (Google Gemini API).

Features

1.Integrates directly into Gmail

2.Reads incoming emails to understand context

3.Sends request to backend API with email content

4.Backend calls Gemini API to generate professional email replies

5.Auto-fills the generated email in Gmail compose box

6.Supports user instructions like tone, style, and format


Technologies Used

1.Chrome Extension (JavaScript )

2.Backend: Spring Boot (java 24)

3.Google Gemini API for AI email generation

4.RESTful API for communication between extension and backend
